## 山猫管理后台前端框架

#### 目录说明

```
├── public # 公共静态资源目录
├── src # 主目录
│   ├── api # 接口文件
│   ├── assets # 资源文件
│   │   ├── images # 图片文件夹
│   │   └── css # 样式文件
│   ├── components # 公共组件
│   ├── design # 样式文件
│   ├── layouts # 布局文件
│   │   └── page # 页面布局
│   ├── main.ts # 主入口
│   ├── router # 路由配置
│   ├── store # 全局状态管理
│   ├── utils # 工具类
│   └── views # 页面
├── webpack # webpack运行及打包
├── vite.config.ts # vite配置文件

```
